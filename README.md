# Engineering Knowledge Base

A personal technical knowledge base built with Next.js, TypeScript, Tailwind CSS, and Markdown.

## Purpose

This project organizes software engineering knowledge by categories such as languages, frameworks, architecture, cloud, infrastructure, databases, AI, and DevOps.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Markdown
- React Markdown
- Remark GFM
